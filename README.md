# about
sixsev'inate the word to appear in a six seven pattern

this is a something i originally made to be one of the cool kid when playing discord wordle

# compile 
- clone this git
```sh
git clone https://github.com/hxero/67-nator
cd 67-nator
```
- then do something
```sh
./sixseven hello
```

- alternatively you can recompile yourself
```sh
gcc -o sixseven main.c 
```
or
```sh
clang main.c -o sixseven
```

# usage
```sh
./sixseven <word>
```
